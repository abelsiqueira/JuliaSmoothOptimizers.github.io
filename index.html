Julia Smooth Optimizers

Here you will find a collection of Julia packages for Nonlinear Optimization. These packages provides all the tools needed for creating optimization problems, and optimization methods. Eventually, we'll provide pure Julia implementations of Large-Scale robust methods.

  • AMD.jl: Approximate Minimum Degree Ordering in Julia
  • AmplNLReader.jl: Julia AMPL Models Conforming to NLPModels.jl
  • CUTEst.jl: Julia's CUTEst Interface
  • HSL.jl: Julia interface to the HSL Mathematical Software Library
  • Krylov.jl: A Julia Basket of Hand-Picked Krylov Methods
  • LinearOperators.jl: Linear Operators for Julia
  • mp: An open-source library for mathematical programming
  • MUMPS.jl: A Julia Interface to MUMPS
  • NLPModels.jl: Data Structures for Optimization Models
  • OptimizationProblems.jl: Optimization Problems for Julia
  • Optimize.jl: Optimization Algorithms in Pure Julia
  • Profiles.jl: Performance and data profiles
  • PROPACK.jl: Julia wrapper of the PROPACK sparse SVD library
  • qr_mumps.jl: Interface to multicore QR factorization qr_mumps
